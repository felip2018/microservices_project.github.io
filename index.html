Actividad #3 - Ingeniería Web II

El actual proyecto tiene como propósito evidenciar un ejemplo del patrón de arquitectura orientado a microservicios.

Presentamos dos servicios que consisten en:

  1. Un servicio POST para una validación de inicio de sesión /uniminuto_services/login
  2. Un servicio GET para una consulta de pokemones /uniminuto_services/get-pokemons

Ingresa el usuario admin y clave 123 para validar el inicio de sesión

Da clic en Consulta pokemones para obtener la lista de pokemons
